# RFC-016 — Variable Constellation Anchors and Canonical Why Fallback

## Graph

Constellation templates also define the Question anchor. Depending on the
stable selected template, Question may sit to the left, above, below, or beside
the first reasoning constellation. It remains the first revealed node and the
root of its flow edges, but is not locked to one screen side.

## Report

When neither conclusion rationale nor Judge synthesis is present, `Why` is a
concise canonical summary of evidence count, perspective count, unresolved
conflict status, and overall confidence. It does not claim an unrecorded Judge
decision.

No session schema, backend workflow, graph semantics, or replay order changes.
