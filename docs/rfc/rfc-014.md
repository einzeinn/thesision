# RFC-014 — First-Read Graph and Concise Report Refinement

## Graph

Constellation geometry supplies the organic visual form. All visible graph
edges are straight line segments so the first reading emphasizes node order and
local relationships rather than curved routing.

## Report

- Repeated evidence claims are rendered once as a shared retrieved finding;
  every source retains its own titled source focus and link.
- `Decision Would Change If` uses a compact HTML details/summary block in
  Markdown so the count is visible and conditions expand on demand.
- The deterministic report uses conclusion rationale, then Judge synthesis, as
  the canonical `Why` fallback. Trade-offs use their recorded list, then the
  recorded perspective analysis.

No source-specific fact is inferred when the grounded provider returns only a
title and URL; the report labels that source focus rather than fabricating a
claim.
