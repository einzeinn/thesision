RFC-008

Canonical Export Pipeline

Objective

JSON menjadi single source of truth.

Markdown tidak boleh membangun ulang reasoning.

Markdown hanya merender JSON.

Requirements

- Export Markdown membaca exported session state.

- Tidak membaca runtime state.

- Tidak membaca UI state.

- Tidak melakukan summarization.

- Tidak melakukan truncation.

- Semua node dirender sesuai JSON.
