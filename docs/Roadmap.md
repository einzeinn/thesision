# 09 - Roadmap

# Thesision

> Reason Before Decision.

---

# Philosophy

Build the smallest product that demonstrates transparent AI reasoning.

Every milestone should move Thesision closer to becoming a trustworthy AI reasoning platform.

---

Planning

Status

Completed

Deliverables

- Product Principles
- Vision
- PRD
- Architecture
- Technical Decisions
- Hackathon Strategy

---

# Phase 1

Foundation

Goal

Build the minimum working architecture.

Tasks

- Project initialization
- Frontend setup
- Backend setup
- OpenAI integration boundary and environment configuration
- Session management
- API communication

Success

The application can receive a question, persist a session, and has a configured boundary for the reasoning provider. Full reasoning execution belongs to Phase 2.

---

# Phase 2

Reasoning Engine

Goal

Implement the AI workflow.

Tasks

- Hypothesis Generator
- Evidence Retrieval
- Perspective Agents
- Judge
- Confidence Evaluation
- Orchestrator

Success

The reasoning pipeline works end-to-end.
